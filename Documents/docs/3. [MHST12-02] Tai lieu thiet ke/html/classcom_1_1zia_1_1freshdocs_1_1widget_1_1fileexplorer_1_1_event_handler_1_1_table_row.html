Detailed Description

A nested class to handle displaying a custom view in the ListView that is used in the Main activity. If any icons are to be added, they must be implemented in the getView method. This class is instantiated once in Main and has no reason to be instantiated again.

void com.zia.freshdocs.widget.fileexplorer.EventHandler.TableRow.killMultiSelect ( boolean  clearData)

This will turn off multi-select and hide the multi-select buttons at the bottom of the view.

Parameters
clearDataif this is true any files/folders the user selected for multi-select will be cleared. If false, the data will be kept for later use. Note: multi-select copy and move will usually be the only one to pass false, so we can later paste it to another folder.
