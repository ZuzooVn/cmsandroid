Detailed Description

This class is completely modular, which is to say that it has no reference to the any GUI activity. This class could be taken and placed into in other java (not just Android) project and work.

This class handles all file and folder operations on the system. This class dictates how files and folders are copied/pasted, (un)zipped renamed and searched. The EventHandler class will generally call these methods and have them performed in a background thread. Threading is not done in this class.
